ТЕХНИЧЕСКИЙ СТЕК И ИНФРАСТРУКТУРА ТЕХНОПРОГРЕСС

ЯЗЫКИ ПРОГРАММИРОВАНИЯ:
- Python 3.12 - основной язык для backend и ML
- Go 1.22 - высоконагруженные сервисы
- TypeScript - frontend разработка
- SQL - работа с базами данных

ФРЕЙМВОРКИ:
- FastAPI - REST API на Python
- Django - административные панели
- React 18 - веб-интерфейсы
- Gin - HTTP сервисы на Go

БАЗЫ ДАННЫХ:
- PostgreSQL 16 - основная реляционная БД
- Redis 7 - кэширование и очереди
- MongoDB - документоориентированные данные
- ClickHouse - аналитика и логи
- Qdrant - векторная база данных для AI

ИНФРАСТРУКТУРА:
- Kubernetes - оркестрация контейнеров
- Docker - контейнеризация
- GitLab CI/CD - непрерывная интеграция
- Terraform - Infrastructure as Code
- Prometheus и Grafana - мониторинг

МЕТОДОЛОГИИ:
- Agile/Scrum - спринты по 2 недели
- Code Review обязателен перед мержем
- Покрытие тестами минимум 80%
- Документация в Confluence
- Задачи в Jira

БЕЗОПАСНОСТЬ:
- SOC 2 Type II сертификация
- Шифрование данных AES-256
- Двухфакторная аутентификация
- VPN для удаленного доступа
